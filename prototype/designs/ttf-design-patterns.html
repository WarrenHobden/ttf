01

How the system is built.

Read this first

Colours are grouped by role, not by hue. A colour's role tells you where it is allowed to appear — and, crucially, whether it may carry text. Several FRANK accents are vivid enough for a fill or an icon but too light to pass as text; the tables below say so plainly so the decision is never left to chance.

Brand

Blue leads, teal grounds

FRANK Blue is the hero — links, buttons, interactive states. Deep Teal is the dark anchor for headers, panels and the wordmark. Teal is the quieter partner.

Accents

Expressive, used sparingly

Pink for emphasis and alerts; green, yellow, cyan and the rest for state and category. Each has a defined job and a defined contrast ceiling.

Neutrals

Grey does the reading

Body grey #4A4A4A carries almost all running text. Mid grey is for secondary copy; the palest grey is for lines and dividers only — never text.

Brand & primary.

Links · buttons · dark surfaces

The dominant non-neutral on the page is a strong blue, defined as both --primary and --blue. Deep Teal is the dark brand accent that carries most panels and the logo. One caution worth designing around: FRANK Blue clears AA as text on white by a whisker, so it should sit on white or near-white — not on the grey surface.

08

Ready-to-paste · CSS variables.

Drop into :root

The complete token set as custom properties. Paste into :root and reference with var(--token). Names describe the job; the two pink tokens keep the readable pink separate from the display pink.

/* ================================================ TALK TO FRANK · COLOUR TOKENS · v1 WCAG 2.2 AA verified ================================================ */ :root { /* BRAND / PRIMARY */ --frank-blue: #0072EE; /* links, CTAs — AA on white */ --deep-teal: #0E3944; /* dark surface, wordmark */ --teal: #235461; /* ACCENTS */ --pink: #FC016A; /* display / fills only */ --pink-text: #D10058; /* pink that carries text */ --green: #78FF74; /* fill only + dark ink */ --yellow: #F2C94C; /* fill only + dark ink */ --indigo: #6610F2; --purple: #CA24C2; --cyan: #17A2B8; /* large / fill */ --orange: #FD7E14; /* fill only + dark ink */ --red: #DC3545; /* INK / NEUTRALS */ --ink: #0B0C0C; /* GDS text tone */ --body: #4A4A4A; /* main body text */ --gray-dark: #343A40; --mid: #767676; /* secondary — on white only */ --line: #ADB5BD; /* borders/dividers — non-text */ /* SURFACES */ --white: #FFFFFF; --light: #F0F4F5; --pink-tint: #FFEEF5; }
